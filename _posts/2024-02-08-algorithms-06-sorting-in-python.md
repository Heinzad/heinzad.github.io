Algorithms: Sorting in Python 
============================= 

Sorting methods are classified as: 
* quadratic - O(n2) 
* logarithmic - O (n log n) 

## Quadratic sorting methods

* Bubblesort
* Selection sort
* Insertion sort
* Shell sort

## Logarithmic sorting methods 

* Quicksort 
* Mergesort
